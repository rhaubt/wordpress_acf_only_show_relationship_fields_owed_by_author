TITLE
--------------------------
ACF - Only Show ACF Relationship Fields for Posts owned by Author


DESCRIPTION
--------------------------
This tool allows to filter Posts and only Show ACF Relationship Fields / Posts Owned by Author.



INSTRUCTIONS
--------------------------
Use Snippet in WordPress functions.php in your child theme folder.

add name=acf field name

See functions.php for more details.
